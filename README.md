# GAIL - a garden irrigation system

A Home Assistant Blueprint and script for automations that operate a multi-zone irrigation system based on moisture sensors and the weather forecast.
Includes hardware tips and full Home Assistant setup instructions. 

For details see the [Wiki pages](https://github.com/AndySymons/GAIL-Garden-irrigation-system/wiki)
